Для создания схемы базы данных потребуется СУБД PostgreSQL версии 9.

1. Установите СУБД PostgreSQL на ваш компьютер.
2. Создайте роль для владельца баз ыданных.
3. Выполните скрипт для создания объектов базы данных netstat.sql.
4. Скопируйте файл "liquibase.properties" в корень проекта.
5. Отредактируйте параметры файла таким образом, чтобы они соответствовали вашим учетным данным.
6. Соберите проект Maven-ом для вступления в силу всех изменений.
